# Запуск приложения

* Для запуска приложения требуется скачать проект
* Затем перейти во вкладку gradle->tasks->build->build
* После запустить docker-compose в нём находится база данных и pgAdmin
* Миграция с производителями уже сделана в приложении на данный момент их всего 4
"dexp", "apple", "hp", "alienware"
* Затем необходимо запустить приложение через класс Application 
* В вашем браузере проходите по ссылке http://localhost:8080/swagger-ui/index.html
здесь будут доступны все контроллеры
